# JapaneseOnlineInterviewSimulator
JapaneseOnlineSimilatorは、パソコンを操作して、オンライン面接を疑似体験するためのシリアスゲームです。

# ゲームの流れ
ゲームが起動すると、スタート画面が表示されます。画面をクリックすると、履歴書の記入に移ります。

履歴書の記入では、名前を入力したり、志望動機を選択します。すべての項目の入力が終わると、面接に移ります。

面接では、面接官があなたに質問してきます。選択肢の中から、あなたが思う最も適切な項目を選択してください。
面接が終わると、リザルト画面に移ります。

リザルト画面では、それぞれの質問に対してのフィードバックが表示されます。

# 操作方法
現状では、基本的にマウスとキーボードのみで操作を行います。

履歴書の記入画面では、名前の入力にキーボード、志望動機やアピールポイントの選択にマウスを使用します。

面接画面では、メッセージを進めたり、選択肢を選ぶときにマウスを使用します。

# 注目ポイント
- 美少女に圧迫面接をしてもらえる点
- 声から意図を汲み取る練習ができる点
- バーチャル背景やSNS通知など、オンライン面接特有のポイントを学べる点
- 質問に対するフィードバックやアドバイスがあり、面接対策にもなる点
